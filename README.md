# Arduino IR Remote Relay Controller

An Arduino-based **Infrared (IR) Remote Relay Control System** developed as a semester project for the **Digital Logic Design Laboratory**.

The project demonstrates wireless appliance control using an infrared remote, an IR receiver, and relay modules, providing a simple and cost-effective home automation solution.

---

## 📌 Project Overview

The system receives commands from a standard infrared remote control through an IR receiver connected to an Arduino Nano.

Based on the received command, the Arduino activates or deactivates relay switches that can control electrical appliances such as:

- Lights
- Fans
- Lamps
- Other low-power electrical devices

## ⚙ Working Principle

1. The IR receiver detects signals transmitted by a standard TV remote.
2. Arduino decodes the received IR command.
3. Based on the button pressed, the corresponding relay is switched ON or OFF.
4. The relay controls connected electrical loads such as lights or other appliances.
